# Graphql Server LookupAPI

## Development Setup

### 1. Clone the repository

```bash
$ git clone {repository url}
```

### 2. Install package depnedencies

```bash
$ yarn install # or $ npm install
```

### 2. Start the server

```bash
$ yarn start # or $ npm run start
```

### 3. Open your browser

```
http://localhost:8080/graphql
```
